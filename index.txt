COMANDOS GIT

Configuração inicial:

  git config --global user.name "Condorito Silva"
  git config --global user-email "email"

Iniciando o git

  git init

Comandos git

 git status
 git add <filename>
 git commit -m "Descrição das alterações"
 git log
 git diff

Navegar entre versões

  git checkout

Desfazer alterações

 git clean -f
 
Voltar para o estado da ultima versão
arquivos não commitados

 git reset --hard HEAD~~0

TAG utilizado para nomear os ponteiros de versões

 git tag nomedataga

É necessário fazer checkout para a versão a qual deseja normar


Branch é uma timeline paralela e server para por exemplo 
trabalhar uma versão de correção enquanto a versão de produção
esta na branch principal
 
 git branch nomedanovabranch
 git checkout nomedabranch que deseja navegar
 git branch - mostra quais branchs existem e qual branch vc esta no momento
